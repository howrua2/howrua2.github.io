Method 2: Using ES File Explorer

Another effective method of installing Kodi on Firestick without a computer is by using ES File Explorer. With this method, you can easily download Kodi on your device following these steps:

  1. Download and install ES File Explorer on your Firestick

  3. Click on the ‘New’ button and enter https://kodi.tv/download
  4. Select the Android version of Kodi you want to install and wait for the download to complete

  6. Wait for Kodi to install on your Firestick

Once the installation is complete, you can open Kodi on your Firestick and access all the content you want.
