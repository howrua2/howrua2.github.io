Requirements to Install Kodi on Playstation 4

1. PlayStation 4 console with 5.05 firmware or any earlier version

2. USB Flash Drive (minimum 1GB)

3. Computer or laptop to download files.

The installation process for Kodi on PS4 involves multiple steps to complete successfully:

How to Install Kodi on Playstation 4?

First, you need to download Kodi on your computer or laptop. Ensure you download the stable version of Kodi. Once downloaded, follow the instructions mentioned below:

Step 1: Plug in the USB drive and format it to FAT32

Step 2: Create a new folder in the USB named “PS4” and inside that create another folder named “UPDATE”.

Step 3: Now, download the latest custom firmware called “PS4 Exploit Host” from the official website.

Step 4: Extract the PS4 Exploit Host on your computer and copy the folder “PS4” into the root directory of your USB drive.

Step 5: On your PS4 console, go to “Settings” - “System Software Update.”

Step 6: Select “Update from USB Storage Device” and select “OK”.
