Kodi is a popular open-source media player that lets you play media files, stream online content, and access various add-ons. One such add-on that is popular among Kodi users is Fusion. Installing Fusion on Kodi 17.3 can be a bit tricky, but it’s worth it because Fusion offers a wide range of video add-ons and features that are not available in Kodi’s official repository. In this article, we’ll provide you with a step-by-step guide on how to install Fusion on Kodi 17.3.

Pain Points of Installing Fusion on Kodi 17.3

One of the biggest pain points of installing Fusion on Kodi 17.3 is that it is not available in Kodi’s official repository. This means that you need to install it from a third-party repository, which can be a bit daunting for some users. Additionally, if you don’t install Fusion correctly, it can cause issues with other add-ons and repositories on Kodi.

How to Install Fusion on Kodi 17.3

The first step to installing Fusion on Kodi 17.3 is to allow unknown sources in Kodi’s settings. You can do this by going to Settings > System Settings > Add-ons and toggling on the slider for Unknown sources. Once you’ve done that, you can follow the steps below:

  1. Open Kodi 17.3 and go to Settings > File Manager.
  2. Click on Add Source and type in the URL for Fusion: http://fusion.tvaddons.ag and click OK.
  3. Next, go back to the Kodi home screen and click on Add-ons.
  4. Click on the Package Installer icon (it looks like an open box).
  5. Select Install from Zip File and navigate to Fusion > begin-here.
  6. Click on the repository zip file and wait for the installation to complete.
  7. Next, go back to the Package Installer and select Install from Repository.
  8. Select TVADDONS.CO Addon Repository and click on Video Add-ons.
  9. Select the add-ons you want to install and click on Install.
  10. Wait for the installation to complete and you should see the add-ons in your Video Add-ons section.

Summary of Installing Fusion on Kodi 17.3

Installing Fusion on Kodi 17.3 involves adding the Fusion source to Kodi, installing the Fusion repository, and installing the video add-ons you want.
