How to Install Indigo on Kodi 17

The first step in installing Indigo on Kodi 17 is to add the Fusion source to Kodi. To do this, follow these steps:

  1. Open Kodi and click on the gear icon in the top-left corner to access the System menu.
  2. Select File Manager.
  3. Click on Add Source.
  4. Select None and copy and paste the following URL: http://fusion.tvaddons.co
  5. Name the source “Fusion” and click OK.
  6. Go back to the Kodi home screen and click on Add-ons from the left-hand menu.
  7. Select the open-box icon in the top-left corner to access the Add-on browser.
  8. Select Install from zip file.
  9. Select Fusion, and then select the folder named Begin-here.
  10. Click on the file named plugin.program.indigo-x.x.x.zip and wait for the installation to complete.

Once the installation is complete, you can access Indigo from the Add-ons menu. Simply click on Indigo and explore all of its features and benefits.
