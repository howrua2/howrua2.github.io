How to Install Pulse on Kodi 17

First, you need to add a new source location in Kodi. Navigate to the System menu, then select File Manager. Click on Add Source, then enter the source URL. Next, return to the Kodi home screen and go to Add-ons. Select the open box icon, then select Install from Zip File. Choose the source you added earlier and wait for the repository to be installed. After that, navigate to Install from Repository, select the Pulse CCM repository, then select Program Add-ons. Finally, select Pulse Wizard and click Install.
