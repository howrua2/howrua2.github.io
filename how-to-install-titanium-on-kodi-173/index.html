How to Install Titanium on Kodi 17.3: Step by Step Guide

When I first tried to install Titanium on Kodi 17.3, I was intimidated by the installation process. However, once I got started, I found that it was more straightforward than I expected, even as someone new to Kodi.

To install Titanium on Kodi 17.3, follow these steps:

  1. Open Kodi and click on the gear icon to go to Kodi’s settings.
  2. Click on System and then click on Add-ons.
  3. Toggle the slider next to “Unknown sources” to turn it on.
  4. Click Yes when prompted to confirm that you want to allow unknown sources.
  5. Return to the Kodi settings screen and click on File Manager.
  6. Click on Add source and then click on <None>.
  7. Type in the following URL: http://repo.supremebuilds.com/ and click OK.
  8. Name the media source something memorable like Supreme and click OK.
  9. Return to the Kodi home screen and click on Add-ons.
  10. Click on the open box icon and then click on Install from zip file.
  11. Click on the Supreme media source and then click on the file named repository.supremebuilds-1.0.3.zip.
  12. Wait for the add-on installed notification to appear in the top right corner.
  13. Click on Install from repository and select the Supreme Builds Repository.
  14. Select the program add-ons category and click on Supreme Builds Wizard.
  15. Click install and wait for the add-on installed notification.
  16. Open the Supreme Builds Wizard and select the Titanium build.
  17. Click on “Fresh Install” to install the Titanium build onto Kodi 17.3.
  18. Wait for the build to install, and then force close Kodi by unplugging your device and plugging it back in.
