How to Install Castaway on Kodi

Installing Castaway on Kodi is a simple process that should only take a few minutes. Here are the steps you need to follow:

Step 1: Enable Unknown Sources

Before you can install any addons on Kodi, you need to enable unknown sources. This is a setting that allows you to install addons that are not part of the official Kodi repository. To enable this setting, go to Settings and then click on System Settings. From there, click on Add-ons and then enable Unknown Sources.

Step 2: Add the Castaway Repository

Next, you need to add the Castaway repository to Kodi. To do this, go to the Kodi home screen and click on the Gear icon. From there, click on File Manager and then Add Source. In the box that pops up, enter the following URL: http://kdil.co/repo. Give the source a name, such as “Castaway Repository”, and click OK.

Step 3: Install the Castaway Addon

Now you can install the Castaway addon from the repository you just added. Go back to the Kodi home screen and click on Add-ons. From there, click on the box icon in the top left corner and then click on Install from zip file. Next, find the repository you just added and click on it. Finally, click on the file named “repository.castaway-x-x-x.zip”. Wait for the addon to install.

Step 4: Enjoy Castaway!

After the Castaway addon is installed, it will be available in your Kodi addons list. Simply click on it to open it and start streaming all your favorite sports, movies, and TV shows!
