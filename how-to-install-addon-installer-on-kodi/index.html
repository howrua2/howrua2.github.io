How to Install Addon Installer on Kodi

There are different methods to install addon installer on Kodi, but we’ll cover the two most popular ones here:

Method 1: Install Addon Installer from ZIP File

The first method involves downloading the Addon Installer ZIP file and installing it on Kodi. Here are the steps:

  1. Go to ‘System’ > ‘Settings’ > ‘Add-ons’ > ‘Install from zip file’
  2. Navigate to the folder where you saved the ZIP file and select it
  3. Wait for the notification that the Addon Installer has been installed

Now, you’ll find the Addon Installer in your Programs section of Kodi.

Method 2: Install Addon Installer from Fusion Repository

The second method involves installing the Fusion repository and then installing the Addon Installer from there. Here’s how:

  1. Go to ‘System’ > ‘Settings’ > ‘File manager’ > ‘Add source’
  2. Enter the following URL: http://fusion.tvaddons.ag/ and give it a name like ‘Fusion’
  3. Go to ‘System’ > ‘Settings’ > ‘Add-ons’ > ‘Install from zip file’
  4. Select the ‘Fusion’ source
  5. Select ‘xbmc-repos’ > ‘English’ > ‘repository.xbmchub-x.x.x.zip’
  6. Wait for the notification that the repository has been installed
  7. Go to ‘System’ > ‘Settings’ > ‘Add-ons’ > ‘Install from repository’ > ‘TVADDONS.ag Addon Repository’ > ‘Program add-ons’
  8. Find and select Addon Installer
  9. Click ‘Install’
  10. Wait for the notification that Addon Installer has been installed

Now, you’ll find the Addon Installer in your Programs section of Kodi.
